Este proyecto corresponde a la materia de {\bfseries{Compiladores}}. ~\newline
 El objetivo es {\bfseries{diseñar e implementar un analizador léxico utilizando un Autómata Finito Determinista (AFD)}}. ~\newline


El analizador recibirá un archivo fuente escrito en un mini-\/lenguaje definido por el equipo y generará como salida\+: ~\newline

\begin{DoxyItemize}
\item Lista de tokens reconocidos (con lexema, tipo de token y número de línea). ~\newline

\item Errores léxicos encontrados (tipo de error, lexema y línea). ~\newline

\item Tabla de símbolos (con identificadores y las líneas donde aparecen). ~\newline

\end{DoxyItemize}
